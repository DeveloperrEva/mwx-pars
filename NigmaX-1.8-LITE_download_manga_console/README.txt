NigmaX 1.8-LITE by CrazyXacker

Для чего нужно это приложение?

NigmaX - "дополнительное" приложение для MangaWatcherX, основная цель которого - помощь в написании/отладке парсеров,
их тестирование и создание каталогов для MWX. C версии 1.6 обзавелась возможностью массового скачивания любого контента
при наличии парсера. На данный момент 98% всех парсеров заточены на скачивание сканов манги и комиксов.

Lite версия не умеет создавать каталоги. Только скачивание и тестирование парсеров!

Первоначальная подготовка:
Нужно установить JRE 8 (Java Runtime Environment) для вашей системы. 
(сборка проводилась на x64 JDK. На х32 не тестировалась. В теории, должно запускаться на JRE 7)

Как использовать:

Запускать из консоли из папки приложения командой java -jar NigmaX-1.8-LITE.jar
После запуска вы увидите справку по приложению (основные команды).

Изначально с приложением уже есть папка parsers с двумя парсерами: ReadManga и MintManga. С их помощью сразу же можно начать
скачивание нужных сканов. Если вы хотите использовать другие сайты - возьмите нужные парсеры из папки /MangaWatcherX/parsers
вашего телефона и переместите их в папку parsers Нигмы. Важно использовать только эту папку, так как нигма при массовом
скачивании использует авто-поиск парсера и главное место поиска - папка parsers. Еще очень важно, чтобы в имени парсера
был указан домен сайта без домена первого уровня (любые другие символы тоже допускаются. важно наличие домена. Пример:
readmanga.json, readmanga_v3.json, superreadmanga.me.json - допускается. manga-site-read.json - не допускается).

Скачивание файлов производится в папку /download (в корневой папке Нигмы). Дальше создаются папки по такому алгоритму: 
/download/{название сайта и json файла}/{название манги/комикса, которая качается}/{название главы}/{сканы}
При скачивании глав отдельно алгоритм такой: 
/download/{название сайта и json файла}/{uniq (уникальный id) главы}/{сканы}

Основные команды:
download {url/file} 		- скачать ВСЕ сканы указаной манги/комикса
downloadchapter {url/file} 	- скачать указанную ГЛАВУ манги/комикса

Дополнительные флаги:
Тесты:
--all          	- протестировать все парсеры (json файлы) из текущей папки
-skipauth		- пропуск авторизации на сайте. На данный момент авторизация нужна только для сайтов Хентай-тян!, Яой-тян!,
					Nude-Moon и AComics. В остальных случаях - смело используйте этот флаг
-skipmlctest 	- пропуск теста [manga_list_complete]. При скачивании сканов бесполезно
-skipmctest 	- пропуск теста [manga_complete], [chapter_complete], [page_complete]. Позволяет пропустить указанные тесты 
					перед началом загрузки. Полезно при массовом скачивании по ссылкам из файла ибо на данный момент тест 
					проводится всегда перед началом загрузки новой манги/комикса/главы. НО! Вы должны провести один ручной
					тест перед масовым скачиванием. В противном случае - использовать на свой страх и риск!
Полезные штуки:
-colored 		- цветной вывод текста в консоль. ВНИМАНИЕ! Корректно работает на Windows 10 build > 10560, Linux и MacOS
					Если у вас Windows 10 10260 и ниже - используйте ConEmu для адекватного вывода инфы в консоль с этим
					флагом, да и вообще.
-skipcredits 	- пропуск загрузки сканов, которые содержат в имени слово "credit". Полезно, если вам нужны более чистые
					сканы, а переводчики адекватно назвали свои Credits страницы.
-gencomicinfo   - генерация ComicInfo.xml с информацие о скачиваемой манге/сканах. Нужно для создания cbz/cbr архивов или для
					конвертации таких архивов через KCC/HCC
					

Примеры использования:
Вместе с приложением предоставляется три cmd файла для более быстрого старта под Windows.

download.cmd
java -jar NigmaX-1.8-LITE.jar        -colored             -gencomicinfo             -skipauth                 download                  download.txt
|       запуск Нигмы         | цветной вывод текста | генерация ComicInfo.xml | пропуск авторизации | скачивание ВСЕХ глав манги | файл с ссылками на мангу |

downloadchapter.cmd
java -jar NigmaX-1.8-LITE.jar -colored -skipauth       downloadchapter         downloadchapter.txt
												 | скачивание главы манги | файл с ссылками на главы |

P.S. Между java -jar NigmaX-1.8-LITE.jar и download|downloadchapter можете спокойно вставлять нужные вам флаги. Главное, чтобы
они были окружены пробелами.

test_all.cmd												 
java -jar NigmaX-1.8-LITE.jar                 --all                                     -d
					         | тестировать все json файлы из этой папки | вывод результатов тестов в консоль |
							 
P.S. Файл уже настроен для теста всех парсеров из папки parsers.
